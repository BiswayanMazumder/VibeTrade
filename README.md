# 📈 Vantedge | AI-Powered Market Intelligence

Vantedge is a high-performance, Gen-Z inspired fintech dashboard that provides real-time market insights, professional 12-month price targets, and AI-driven sentiment analysis. 

Built with a focus on **Glassmorphism UI** and **Serverless Python**, Vantedge bridges the gap between complex financial data and intuitive, aesthetic design.

## ✨ Key Features

- **🎯 AI Oracle Insight:** Real-time long-term vs. short-term prediction engine based on 1-year annualized returns and trend slopes.
- **📊 Interactive Pro Charts:** Google Finance-style spline charts with "Previous Close" baselines and auto-scaling Y-axes.
- **🔍 Smart Search:** Real-time ticker suggestions and company name lookups powered by Yahoo Finance API.
- **📰 News Pulse:** Live headline fetching to keep users updated on the "why" behind price movements.
- **📐 Fundamental Grid:** Detailed market metrics including P/E Ratio, Market Cap, 52-Week High/Low, and Dividend Yields.
- **🧊 3D Glassmorphism UI:** Premium aesthetic featuring 3D hover effects, blur backgrounds, and neon accents.

## 🛠️ Tech Stack

- **Backend:** [FastAPI](https://fastapi.tiangolo.com/) (Python)
- **Data Engine:** [yfinance](https://github.com/ranaroussi/yfinance), [Pandas](https://pandas.pydata.org/), [NumPy](https://numpy.org/)
- **Visuals:** [Plotly.js](https://plotly.com/javascript/), [Tailwind CSS](https://tailwindcss.com/)
- **Hosting:** [Vercel](https://vercel.com/) (Serverless Functions)
